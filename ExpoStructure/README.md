# 🚀 ExpoStructure

A production-ready **Expo** boilerplate with TypeScript, Expo Router, and support for Android, iOS, and Web.

## Tech Stack

| Technology | Version |
|---|---|
| Expo SDK | 56.0.12 |
| React Native | 0.85.3 |
| React | 19.2.3 |
| TypeScript | 6.0.3 |
| Expo Router | 56.2.11 |

## Quick Start

```bash
git clone https://github.com/darshan8699/RN_Basic_Structure.git
cd RN_Basic_Structure/ExpoStructure
npm install
npx expo start
```

## Run Commands

| Platform | Command |
|---|---|
| Android | `npx expo start` → press `a` |
| iOS | `npx expo start` → press `i` |
| Web | `npx expo start` → press `w` |
